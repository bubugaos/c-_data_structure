## 重复的子字符串

给定一个非空的字符串，判断它是否可以由它的一个子串重复多次构成。给定的字符串只含有小写英文字母，并且长度不超过10000。

```ini
示例 1:
输入: "abab"
输出: True
解释: 可由子字符串 "ab" 重复两次构成。
```

```ini
示例 2:
输入: "aba"
输出: False
```

```ini
示例 3:
输入: "abcabcabcabc"
输出: True
解释: 可由子字符串 "abc" 重复四次构成。 (或者子字符串 "abcabc" 重复两次构成。)
```

```c++
class Solution {
	public:
		void getNext(int* next, string& s) {
			int j = -1;
			next[0] = j;
			for (int i = 1; i < s.size(); i++) {
				while (j >= 0 && s[i] != s[j+1]) {
					j = next[j]; // 前缀表统一减一的实现方法
				}
				if (s[i] == s[j+1]) {
					j++;
				}
				next[i] = j;
			}
		}
		bool repetedSubStringPattern (string s) {
			if (s.size() == 0) return false;
			int j = -1;
			int next[s.size()];
			getNext(next,s);
			/* for (int i = 0; i < s.size(); i++) {
				cout<<next[i]<<" ";
			} */
			int len = s.size();
			if (next[len-1] != -1 && (len % (len-next[len-1]+1) == 0)) 
			return true;
			else return false;
		}
}; // 可以打印一下next数组，对kmp算法有更深层的了解
```
